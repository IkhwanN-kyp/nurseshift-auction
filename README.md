Nurse Leave Bidding System

A leave bidding system for nurses designed to make holiday scheduling more transparent, fair, and structured. This application simplifies leave requests for nursing staff and helps administrators manage approvals efficiently.

Key Features:

    Tier-Based Bidding System: Set leave priority based on hospital policies.

    Period Management: Admin control over leave request periods.

    Admin Panel: Manage invitation codes, nurse tiers, and block specific dates.

    Report Generation: Automatic data export to Excel for reporting purposes.

Deployment Guide:
Please refer to the PANDUAN_DEPLOY.md file in this repository for complete instructions on setting up Supabase and deploying to Vercel.

Sistem Lelang Cuti Perawat

Sistem lelang cuti untuk perawat agar pengelolaan jadwal libur menjadi lebih transparan, adil, dan terstruktur. Aplikasi ini dirancang untuk memudahkan staf perawat dalam mengajukan cuti dan membantu admin dalam mengelola persetujuan secara efisien.

Fitur Utama:

    Sistem Bidding Berbasis Tier: Pengaturan prioritas cuti sesuai kebijakan rumah sakit.

    Manajemen Periode: Admin dapat mengatur periode pengajuan cuti.

    Panel Admin: Pengaturan kode undangan, tier perawat, dan pemblokiran tanggal tertentu.

    Rekapitulasi: Ekspor data otomatis ke Excel untuk laporan.

Panduan Deployment:
Silakan merujuk ke file PANDUAN_DEPLOY.md di repositori ini untuk instruksi lengkap mengenai setup Supabase dan deployment di Vercel.
